# Nidhesh Soni | Full-Stack Developer Portfolio

A high-performance, minimalist, and brutalist-inspired personal portfolio built with the MERN stack (**MongoDB, Express, React, Node.js**). This project features a custom dark mode, responsive design, a backend-driven contact system, and dynamic asset filtering.

## 🛠️ Tech Stack

### Frontend
- **React.js (Vite)** - Fast, modern UI development.
- **Tailwind CSS** - Utility-first styling for brutalist aesthetics.
- **Framer Motion** - Smooth scroll animations and element transitions.
- **Lucide & React Icons** - For high-quality, scalable iconography.

### Backend
- **Node.js & Express.js** - Robust server-side logic and API routing.
- **MongoDB & Mongoose** - Document-based database for inquiry storage.
- **CORS & Dotenv** - Security and environment configuration.

## ✨ Key Features
- **Brutalist UI:** High contrast, bold 4px borders, and offset hard shadows.
- **Dynamic Dark Mode:** Seamless theme switching with persistent local storage.
- **Signature Logo:** Integration of custom cursive Google Fonts for a personal touch.
- **Portrait Tech-Grey Filter:** Real-time CSS `invert` and `brightness` filters applied to portrait assets in dark mode.
- **Responsive Layout:** Completely optimized for mobile, tablet, and desktop viewing.
- **Environment Driven:** Fully configured for separate deployment of frontend and backend.
